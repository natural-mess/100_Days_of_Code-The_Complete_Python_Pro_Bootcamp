# TinDog Project

A responsive landing page for a fictional dog dating app.

This project focuses on practicing Bootstrap 5 layout patterns and basic custom CSS styling.

## Project Overview

TinDog is a single-page website that includes:

- Hero section with app download buttons
- Feature highlights with inline SVG icons
- Testimonial section with brand logos
- Pricing cards for different plans
- Footer navigation links

## Tech Stack

- HTML5
- CSS3
- Bootstrap 5.3 (via CDN)

## Folder Structure

- `index.html`: Main project page
- `css/style.css`: Custom styling for gradient background and reusable UI helpers
- `images/`: Visual assets used by the page
- `goal images/`: Reference screenshots for the target layout
- `solution.html` and `css/solution.css`: Course solution files for comparison

## How to Run

1. Open the project folder.
2. Open `index.html` in your browser.

Optional (VS Code):

1. Install the Live Server extension.
2. Right-click `index.html` and choose **Open with Live Server**.

## Learning Goals Covered

- Using Bootstrap grid and utility classes
- Building reusable page sections
- Creating card-based pricing layouts
- Combining Bootstrap components with custom CSS
- Implementing animated gradient backgrounds

## Notes

- Bootstrap is loaded from a CDN, so internet access is required for Bootstrap styles.
- The project is static and does not include JavaScript interactivity.
